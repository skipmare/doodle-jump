\doxysubsection{Detailed Description}
Represents a bonus item that can be collected by the player. 

The \doxylink{class_bonus}{Bonus} class is a base class for different types of bonuses, such as \doxylink{class_jetpack}{Jetpack} and \doxylink{class_spring}{Spring}. It tracks the position of the bonus item relative to a platform, and updates its position as the platform moves.
