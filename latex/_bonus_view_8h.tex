\doxysubsection{Detailed Description}
Defines the \doxylink{class_bonus_view}{Bonus\+View} class, responsible for rendering a bonus entity in the game. 

This class is used to create a graphical representation of a \doxylink{class_bonus}{Bonus} entity. It loads textures based on the bonus type and sets the position of the sprite in the game window. If texture loading fails, it uses a fallback shape.
