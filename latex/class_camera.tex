\doxysubsection{Detailed Description}
A class to manage the camera\textquotesingle{}s position and transformation of coordinates. 

The \doxylink{class_camera}{Camera} class handles the camera\textquotesingle{}s position in the game world, updating based on the player\textquotesingle{}s position, and provides methods to transform coordinates between world space and screen space.
