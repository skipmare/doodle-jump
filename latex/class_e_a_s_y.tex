\doxysubsection{Detailed Description}
A derived class representing the \doxylink{class_e_a_s_y}{EASY} difficulty settings. 

This class sets the parameters for the \doxylink{class_e_a_s_y}{EASY} difficulty level, with higher chances of platforms appearing and more frequent bonus chances. The distance between platforms is also smaller compared to harder levels.
