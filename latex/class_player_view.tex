\doxysubsection{Detailed Description}
Handles the visual representation of the \doxylink{class_player}{Player} entity. 

This class manages the rendering of the \doxylink{class_player}{Player} entity\textquotesingle{}s textures, allowing for dynamic switching based on the player\textquotesingle{}s direction (e.\+g., moving left or right). It updates the position and visual state of the \doxylink{class_player}{Player} entity on the screen.

This class is responsible for rendering the \doxylink{class_player}{Player} entity on the screen, including managing textures for different directions (left and right) and a fallback shape in case textures fail to load. It dynamically updates the sprite and position based on the \doxylink{class_player}{Player}\textquotesingle{}s state.
