\doxysubsection{Detailed Description}
An abstract base class for the \doxylink{class_observer}{Observer} design pattern. 

The \doxylink{class_observer}{Observer} class is used to define a contract for objects that wish to be notified of changes in a \doxylink{class_subject}{Subject}. This class defines an update method that will be called when the subject undergoes a change.
