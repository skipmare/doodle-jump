\doxysubsection{Detailed Description}
Class for rendering and updating the view of a platform entity. 

\texorpdfstring{$<$}{<} Include the \doxylink{class_platform}{Platform} class for platform properties Include the \doxylink{class_entity_view}{Entity\+View} class as a base class SFML for rendering graphics

The \doxylink{class_platform_view}{Platform\+View} class is responsible for rendering the graphical representation of a platform, updating its position, and handling texture loading based on the platform type.
