\doxysubsection{Detailed Description}
A class that manages and notifies a list of observers. 

The \doxylink{class_subject}{Subject} class follows the \doxylink{class_observer}{Observer} design pattern, allowing observers to attach and detach themselves from the subject. When the subject\textquotesingle{}s state changes, it will notify all attached observers by calling their update method.
